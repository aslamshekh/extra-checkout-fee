== Description ==

Checkout extra fee plugin gives the option to set up a checkout fee based on payment gateway configuration.

Store owners complete control to adjust Fee Labels, Amount by percentage or fixed price calculation.

This plugin gives you a global setting or payment gateway wise take checkout fee.

Features
---------------------------------------
1) Set a checkout fee based on the payment gateway.
2) Set checkout fee globally for all payment gateway.
3) Set checkout fee fixed price or percentage wise
4) Set a checkout fee label.

== Installation ==

From WordPress Dashboard go to “Plugins >> Add New >> Uploads”, select ‘quick-stock-management.zip’ file and upload it. Or,
Extract the zip file, copy and paste to the ‘wp-conten/plugins/’ directory of your WordPress Installation.
Activate the plugin.
